# Net API

```js
import Net from 'erebos-api-net'
```

### new Net()

Creates an Net instance using the provided `rpc` handler.

**Arguments**

1.  `rpc: RPC`

### .version()

**Returns** `Promise<string>`

### .listening()

**Returns** `Promise<boolean>`

### .peerCount()

**Returns** `Promise<hex>`
